# Fuzz

**Шаблон репозитория для настройки invoke-задач под проект с целью автоматизации создания и запуска скриптов различного назначения**
- конфигурирование, сборку и запуск тестов
- настройку рабочей среды
- различные иные аспекты в зависимости от конкретного проекта

## Как пользоваться:
Два простых шага:
1. Настроить задачи под проект (изменить данный README.md)
0. Из корня директории Fuzz выполнить:
    - На Linux (Shell):
        ```bash
        . start.sh
        ```
    - На Windows (PowerShell):
        ```bat
        start.ps1
        ```

# Дополнительная информация:

## ⚠️ Только Python
Базово, весь функционал данного репозитория написан на Python в концепции использования [invoke задач](https://www.pyinvoke.org/).  
Крайне важно, чтобы в данном репозитории отсутствовали любые иные ЯП, чтобы поддержать простоту и чистоту запуска задач на различных платформах.

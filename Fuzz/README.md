# Fuzz

**Вспомогательный функционал для работы с кодовой базой репозитория:**
- настройка рабочей среды (автоматическая установка всех необходимых программ и библиотек);
- конфигурирование;
- сборка;
- запуск тестов

## Как пользоваться:
1. Из корня директории Fuzz выполнить:
    - На Linux (Shell):
        ```bash
        . start.sh
        ```
    - На Windows (PowerShell):
        ```bat
        start.ps1
        ```

# Дополнительная информация:
## ⚠️ Только Python
Базово, весь функционал данного репозитория написан на Python в концепции использования [invoke задач](https://www.pyinvoke.org/).  
Крайне важно, чтобы в данном репозитории отсутствовали любые иные ЯП, чтобы поддержать простоту и чистоту запуска задач на различных платформах.

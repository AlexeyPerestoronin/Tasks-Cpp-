# Tasks (C++)
**Практика в C++ через решение алгоритмических задач**

# Дополнительная информация
1. Все свспомогательные процессы (развёртывание, сборка, запуск и т.д.) автоматизированы при помощи [Fuzz](./Fuzz/README.md)
0. Для первичного запуска на ОС Windows-10:
    1. `cd ./Fuzz`
    0. `start.ps1`
    0. `invoke leet-code.full-check --detect-conan-profile --target "Two-Sum"`
        - используйте параметр `--detect-conan-profile` только если Conan на вашей ОС будет установлен и использован впервые
        - репозиторий содержит большое количество решённых задач, однако параметр `--target "Two-Sum` для проверки работоспособности запускает только `Two-Sum(.exe)`
            - если необходимо запустить все имеющиеся тесты не используйте данный параметр, однако знайте, что некоторые задачи являются шаблонами, из-за чего целевые тесты не будут проходить
                - шаблонными являются задачи, для которых существует одноимённый MarkDown файл (например `TwoSum.md`), содержащий различные решения целевой проблемы
